## LED Settings
LED Pin - `0`
Max Brightness - `100`
LED Layout  - `WASD`
Brightness Steps - `5`
LEDs per Button - `1`

## LED Order

(From left to right)

```
Up > Down > Left > Right > L1 > R1 > B2 > B1 > B3 > B4 > R2 > L2
```
